# Scheduling Assistant

A voice-controlled web application for creating, managing, and organizing reminders using speech commands. Built with HTML, CSS, and JavaScript, the application provides a clean, responsive interface and utilizes the Web Speech API to enable hands-free scheduling.

## Overview

Scheduling Assistant is designed to simplify daily task management by allowing users to create reminders through voice commands instead of manual input. The application demonstrates modern front-end development concepts while providing an intuitive and user-friendly experience.

## Features

- Voice-controlled reminder creation
- Schedule and organize reminders
- Alarm notification interface
- Snooze and dismiss reminder options
- Responsive and modern user interface
- Browser-based application with no backend required

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Web Speech API
- Tailwind CSS
