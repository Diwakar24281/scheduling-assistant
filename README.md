🎙️ Scheduling Assistant

A modern, voice-controlled scheduling and reminder web application that allows users to create and manage reminders using speech commands. The application provides a simple, responsive interface and leverages the Web Speech API for hands-free interaction.

Overview

Scheduling Assistant is designed to simplify task and reminder management through voice recognition. Instead of manually typing reminders, users can interact with the application using natural voice commands, making scheduling faster and more convenient.

 Features

- 🎤 Voice-based reminder creation
- 📅 Schedule and manage reminders
- ⏰ Alarm and notification interface
- 💤 Snooze reminders
- ❌ Dismiss reminders
- 📱 Responsive and modern user interface
- 🌐 Runs entirely in the browser (No backend required)

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Web Speech API
- Tailwind CSS

 Getting Started

Prerequisites

- A modern web browser (Google Chrome or Microsoft Edge)
- Visual Studio Code (Recommended)
- Live Server Extension (Optional)

Installation

1. Clone the repository

```bash
git clone https://github.com/Diwakar24281/scheduling-assistant.git
```

2. Open the project in Visual Studio Code.

3. Open `index.html` using Live Server or any modern browser.
